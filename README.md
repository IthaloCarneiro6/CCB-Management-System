O **Sistema de Gestão CCB** é uma plataforma inteligente desenvolvida para centralizar e automatizar a logística do **Circuito Cearense de Basquete**. O sistema resolve o desafio complexo de organizar rodadas que envolvem múltiplas categorias, equipes do interior e restrições rigorosas de disponibilidade.

## ⚡ Principais Funcionalidades

*   **Matchmaking Inteligente:** Algoritmo que cruza automaticamente as partidas pendentes com as disponibilidades enviadas pelas equipes para datas específicas.
*   **Gestão de Rodadas Mistas:** Capacidade de planejar rodadas que integram diferentes categorias (como Adulto Masculino Ouro e Adulto Feminino) simultaneamente.
*   **Interface "Modern Sports Dark":** Design de alto contraste em tons de preto e laranja vibrante, focado na usabilidade e performance do gestor.
*   **Importação de Dados (CSV):** Processamento rápido de planilhas de disponibilidade e tabelas de jogos via API integrada.
*   **Controle de Status e Logs:** Monitoramento em tempo real do ciclo de vida dos jogos (Pendente → Aguardando → Realizado) com registro de auditoria.

## 🛠️ Stack Tecnológica

*   **Frontend:** Next.js 15 (App Router), React, Tailwind CSS.
*   **Backend:** Next.js Route Handlers e Supabase (PostgreSQL).
*   **Gerenciamento de Estado:** Zustand e Hooks nativos do React.
*   **Ferramentas:** Lucide React (Ícones), PapaParse (Processamento de CSV).
